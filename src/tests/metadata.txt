SCORING KEY (per criterion, 0-100)
────────────────────────────────────────────────────────────────────────────────
  HIGH (70-100) → evidence of REAL / authentic photograph
  MED  (40-69)  → ambiguous / insufficient signal
  LOW  (0-39)   → evidence of AI-GENERATED / synthetic / manipulated

  overall confidence_score = avg of all 26 criteria scores
  verdict:  >= 60 → "real"  |  < 60 → "ai_generated"

────────────────────────────────────────────────────────────────────────────────
26 FORENSIC CRITERIA (used in breakdown)
────────────────────────────────────────────────────────────────────────────────
 1. texture_consistency      — natural vs. AI-smooth/repetitive textures
 2. lighting_analysis        — physically plausible light sources
 3. edge_detection           — organic vs. unnaturally sharp/soft edges
 4. noise_pattern            — camera sensor noise vs. AI noise-free output
 5. facial_analysis          — real face micro-details vs. GAN symmetry
 6. metadata_integrity       — compression / origin marker authenticity
 7. color_distribution       — natural color histograms vs. AI uniformity
 8. artifact_detection       — malformed text / hands / geometry / seams
 9. shading_analysis         — shading consistent with light and object form
10. depth_of_field           — physically realistic bokeh falloff
11. motion_blur              — blur consistent with implied movement
12. background_consistency   — foreground/background light & style match
13. shadow_analysis          — correct shadow direction and intensity
14. perspective_correctness  — realistic vanishing points and proportions
15. compression_artifacts    — natural JPEG vs. suspicious patterns
16. image_text_indication    — 100 = no text; penalised for AI-garbled text
17. contents_reality         — physically plausible scene content
18. ai_fingerprint           — GAN/diffusion frequency anomalies
19. clone_detection          — repeated/copy-pasted patches
20. scale_proportion         — correct relative object sizes
21. optical_anomalies        — physically plausible reflections/distortions
22. exif_manipulation        — EXIF timestamp/GPS/camera field consistency
23. semantic_consistency     — all elements logically co-exist
24. internet_presence        — found on credible sources vs. AI platforms
25. watermark_integrity      — C2PA/SynthID/steganographic signals
26. image_description_analysis — plausibility score from scene description

HOW TO RUN
────────────────────────────────────────────────────────────────────────────────
1. Open the app → Home page (URL input mode)
2. In browser DevTools console run:
     const { runTests } = await import('/src/test/runner.js');
     runTests();                          // all 30
     runTests({ only: 'real' });          // real only
     runTests({ only: 'ai' });            // AI only
     runTests({ ids: ['t001','t021'] });  // specific
3. Results auto-saved to localStorage["realai_test_results"]
4. Copy console.table output into test/results.txt for tracking
